University of Wollongong in Dubai

Education

This project demonstrates the use of HTML and CSS to design and develop an informational website for the University of Wollongong in Dubai. HTML was used to structure the webpage by organizing content such as headings, navigation menus, images, and text sections. CSS was applied to enhance the visual appearance by adding colors, layouts, fonts, and styling to create a clean and user-friendly design. The website includes well-structured sections such as the header, navigation bar, main content area, and footer. The main goal of this project was to practice web development skills and build a professional-looking, easy-to-navigate university information website while improving understanding of responsive layout design and modern web styling techniques.

Game World

APP

I designed this Game World mobile application interface using Figma as part of my UI/UX design project. The goal of this project was to create a simple and user-friendly gaming store app where users can browse and purchase gaming products. I created multiple screens including a splash screen, login page, account sign-in page, and home page that displays gaming content and categories. I also designed product pages where users can view items such as gaming consoles and accessories. In addition, the design includes a shopping cart, payment page, and order confirmation screen to complete the purchase process. This project helped me practice mobile interface design, layout organization, and user flow planning, while improving my skills in Figma UI design principles.

BookHaven

Social

BookHaven is a web application designed to help users easily discover and explore books online. The project focuses on creating a clean and user-friendly digital library experience. The homepage includes a hero section with a search bar that allows users to quickly search for books, authors, or genres. A side navigation menu provides access to categories such as Fiction, Mystery, Romance, Science Fiction, and more. The website also includes a Featured Books section displaying popular titles with their cover images. The project uses HTML, CSS, and JavaScript for the front end, while the back end is developed using Node.js to manage data and handle user requests, helping improve both front-end and back-end development skills.
